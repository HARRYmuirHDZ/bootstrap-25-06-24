¿Qué es forEach?

El método forEach es una función en JavaScript que se usa para iterar sobre los elementos de un arreglo. Permite ejecutar una función específica en cada elemento del arreglo, proporcionando una manera más sencilla y legible de trabajar con los elementos de un arreglo.

Sintaxis básica con función flecha

La sintaxis básica de forEach utilizando una función flecha es la siguiente:


        Solo escribe en Visual studio Code Foreach ahora reemplaza "array" por tu arreglo "series", cada uno quedará en "item" si posee objetos coloca "item.propiedad"
array.forEach(element => {
    // Código a ejecutar por cada elemento
});

Ejemplo simple

Imagina que tienes una caja de juguetes y quieres mostrar el nombre de cada juguete uno por uno. Vamos a crear un arreglo de juguetes y usar forEach para decir el nombre de cada juguete.


// Arreglo de juguetes
let juguetes = ["Carro", "Muñeca", "Pelota", "Tren"];

// Usamos forEach para decir el nombre de cada juguete
juguetes.forEach(juguete => {
    console.log(juguete);
});

Explicación

  • Arreglo de juguetes: Imagina que tienes una lista con los nombres de tus juguetes favoritos.
  • forEach: Es como decir "Por cada juguete en la lista, haz algo".
  • Función flecha: Es una manera rápida de escribir una pequeña acción que quieres hacer por cada juguete. En este caso, la acción es mostrar el nombre del juguete.
